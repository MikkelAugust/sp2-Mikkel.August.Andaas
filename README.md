## Fortis Auction

A modern auction platform built with HTML, CSS, and JavaScript using the Noroff Auction API.

### Features

- User registration & login
- Create, edit, and delete listings
- Bid on listings
- Profile pages (own & other users)
- Responsive design

### Tech stack

- HTML
- CSS
- JavaScript (ES Modules)
- Noroff Auction API v2

### Setup

1. Clone the repo
2. Open with Live Server
3. Register a @stud.noroff.no user

### Notes

This project is built for educational purposes.
